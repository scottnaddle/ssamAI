# ssamAI Pipeline 아키텍처 — 선생님 워크로드 감소 루프

> **핵심 미션**: 선생님의 행정업무를 최대한 줄여준다.
> 8개 스킬 + 2개 인프라 + 3개 자가 학습 루프로 매일 진화하는 시스템

---

## 1. 시스템 개요

### 1-1. 8대 핵심 스킬 (런타임)

| # | 스킬 | 트리거 | 생성 문서 | 평균 절약 |
|---|------|--------|----------|----------|
| 1 | **lesson-plan-generator** | "수업 과정안", "교수학습과정안" | HWPX (1~2쪽) | 90분/차시 |
| 2 | **formative-assessment-generator** | "형성평가", "퀴즈" | HWPX (10문항 + 정답표) | 60분/세트 |
| 3 | **rubric-generator** | "루브릭", "수행평가 채점기준" | HWPX (3×3~5×4 표) | 40분/평가 |
| 4 | **official-letter-generator** | "공문", "연수보고서", "출장" | HWPX (행정표준) | 30분/문서 |
| 5 | **home-letter-generator** | "가정통신문", "동의서" | HWPX (안내문 + 동의서) | 25분/문서 |
| 6 | **ssamai-pipeline-orchestrator** | (내부) 자동 라우팅 | (5종 조합) | 전체 최적화 |
| 7 | **continuous-research-agent** | 매일 자동 6AM | (메타 — 5종 업데이트) | - |
| 8 | **teacher-template-loop** | 선생님 업로드 시 | (메타 — 5종 강화) | - |

**주간 절약 (선생님 1인)**: 약 12시간 30분
**월간 절약**: 약 50시간
**연간 절약**: 약 600시간 (8주 = 1학기 기준)

### 1-2. 인프라

```
.claude/skills/ + .config/opencode/skills/
├── lesson-plan-generator/
├── formative-assessment-generator/
```

## 2. 3대 자가 학습 루프 (매일 작동)

### Loop 1: 외부 리서치 → 스킬 강화

```
┌─────────────────────────────────────────────────────────┐
│  [외부 템플릿 소스]                                       │
│  ├─ 에듀넷, 차시랑, 금성/미래엔/천재                      │
│  ├─ 정부24, 경기도교육청                                 │
│  └─ 비즈폼, yesform (양식)                               │
└────────────────────┬────────────────────────────────────┘
                     │ daily-research.sh (launchd 6AM)
                     ▼
┌─────────────────────────────────────────────────────────┐
│  [continuous-research-agent]                              │
│  1. 크롤링 (agent-reach)                                  │
│  2. 분류 (5종 카테고리)                                   │
│  3. 패턴 추출 (YAML)                                     │
│  4. 기존 스킬 비교 (diff)                                │
│  5. 업데이트 제안 (사용자 승인 or 자동)                   │
└────────────────────┬────────────────────────────────────┘
                     │ 자동 패치 (5+명 합의)
                     ▼
┌─────────────────────────────────────────────────────────┐
│  [5개 콘텐츠 스킬 강화]                                    │
│  SKILL.md 패치 / 새 템플릿 추가 / 새 차시 유형 추가       │
└─────────────────────────────────────────────────────────┘
```

### Loop 2: 선생님 콘텐츠 → 스킬 강화

```
┌─────────────────────────────────────────────────────────┐
│  [선생님 Library 업로드/편집]                              │
│  ├─ 형성평가 10개 만드심 → 2번 문항 자주 수정             │
│  ├─ 가정통신문 50개 만드심 → "학부모 확인란" 12번 추가    │
│  └─ 루브릭 5개 → 척도 4단계로 통일                        │
└────────────────────┬────────────────────────────────────┘
                     │ teacher-template-loop
                     ▼
┌─────────────────────────────────────────────────────────┐
│  [편집 패턴 분석]                                         │
│  - 추가 섹션 / 세분화 / 난이도 / 표현 / 순서 / 신규 필드   │
│  - 5명 이상 합의 → 자동 스킬 업데이트                      │
│  - 2~4명 → 제안 (사용자 승인)                              │
└────────────────────┬────────────────────────────────────┘
                     │ 자동 반영
                     ▼
┌─────────────────────────────────────────────────────────┐
│  [5개 콘텐츠 스킬 강화]                                    │
└─────────────────────────────────────────────────────────┘
```

## 3. 연계 워크플로 5종

### Workflow 1: 차시별 자료 패키지 (4종 세트)
**트리거**: "이번 주 2차시 자료 전부 만들어줘"
**생성**:
1. lesson-plan-generator → 교수학습과정안 HWPX
2. (pptx-generator) → 차시별 PPT
3. (hwpx) → 활동지 HWPX
4. formative-assessment-generator → 형성평가 HWPX
**절약**: 4시간/차시 → 10분 (**98% 단축**)

### Workflow 2: 현장학습 일괄 (3종)
**트리거**: "6월 25일 현장학습 자료 만들어줘"
**생성**:
1. home-letter-generator → 가정통신문
2. home-letter-generator → 현장체험학습 동의서
3. (선택) lesson-plan-generator → 현장학습 전후 수업 계획
**절약**: 1시간 30분 → 5분 (**95% 단축**)

### Workflow 3: 수행평가 일괄 (2종)
**트리거**: "5학년 국어 설명문 수행평가 채점 기준 만들어줘"
**생성**:
1. rubric-generator → 수행평가 루브릭
2. (선택) formative-assessment-generator → 수행평가 안내문
**절약**: 1시간 → 3분 (**95% 단축**)

### Workflow 4: 단원 마무리 자료 (3종)
**트리거**: "4단원 끝났어. 평가 자료 만들어줘"
**생성**:
1. formative-assessment-generator → 단원 평가
2. rubric-generator → 수행평가 루브릭
3. home-letter-generator → 가정통신문 (성적 통지)
**절약**: 5시간 → 15분 (**95% 단축**)

### Workflow 5: 학기 초 행정업무 일괄 (5종)
**트리거**: "3월 학기 초 자료 일괄"
**생성**: 5종 (계획서 + 가정통신문 + 공문 + 평가계획 + 진단평가)
**절약**: 6시간 → 20분 (**95% 단축**)

## 4. 데이터 흐름 (Phase 2+)

### 4-1. 저장소

| 데이터 | 저장소 | 용도 |
|--------|--------|------|
| 선생님 프로필 | PostgreSQL (Neon) | 학교/학년/과목/개인화 |
| 생성 이력 | PostgreSQL | 5종 스킬 사용 이력, 만족도 |
| 선생님 템플릿 | PostgreSQL + S3 | 업로드 자료, 편집본 |
| 커뮤니티 템플릿 | PostgreSQL + S3 | 공유 자료, 인기도 |
| 패턴 분석 | Neo4j | 패턴 그래프, 빈도 |
| 리서치 결과 | PostgreSQL (research DB) | 외부 자료 메타데이터 |
| 스킬 업데이트 이력 | Git | SKILL.md 변경 이력 |

### 4-2. 실시간 흐름

```
[Library 페이지]
   │ (선생님이 자료 생성)
   ▼
[오케스트레이터] (5종 + 워크플로)
   │
   ▼
[5개 스킬] → [HWPX 생성] → [S3 저장]
   │
   ├─ 만족도 ⭐ → PostgreSQL
   ├─ 편집 비율 → teacher-template-loop
   └─ 인기도 → 커뮤니티 랭킹

[매일 6AM]
   │
   ├─ daily-research.sh → agent-reach → 패턴 추출
   └─ teacher-template-loop → 편집 분석 → 패턴 추출
   │
   ▼
[Neo4j: 패턴 그래프]
   │ 5+ 합의
   ▼
[Git PR: 스킬 업데이트]
```
